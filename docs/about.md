# About

## Purpose
PoseForge is a modern dark-themed web app for visualizing COLMAP 3D reconstructions in the browser. It accepts COLMAP sparse reconstruction data (binary and text formats) and renders point clouds, camera frustums, trajectories, and camera views in an interactive Three.js viewer.
